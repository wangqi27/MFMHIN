# MFMHIN
MFMHIN: A Multi-Information Fusion Model based on Heterogeneous Information Networks for Predicting Synergistic Drug Combinations
This repository contains the source code and data for the paper "MFMHIN: A Multi-Information Fusion Model based on Heterogeneous Information Networks for Predicting Synergistic Drug Combinations".

Directory Structure
data/: Contains the core dataset files required for the model (e.g., drug combinations, similarity matrices, PPI network).
src/: Contains the source code for data preprocessing, feature construction, HIN embedding, and model training/prediction.
results/: Directory where the output files (feature vectors, prediction results) will be saved.
Prerequisites
Install the required Python packages:

pip install -r requirements.txt
Usage
1. Data Preprocessing
First, run the data preprocessing script to extract drug information and process the raw data:

python src/data_preprocessing.py
2. Feature Construction (Part 1 & 2)
Construct the basic feature vectors (Chemical, ATC, Target, Pathway similarities) and vector addition features:

python src/construct_feature_vectors_1.py
python src/construct_feature_vectors_2.py
3. HIN Embedding (HIN2Vec)
The HIN embedding is performed using the HIN2Vec module located in src/HIN2Vec.
You need to generate the node vectors and metapath vectors.

Note: The file node_vectors_all_l5w3d100.txt is required for the subsequent steps. This file is generated by running the HIN2Vec algorithm on the constructed heterogeneous network. Please refer to src/HIN2Vec/README.md (if available) or the HIN2Vec documentation for detailed instructions on running the embedding.

After generation, ensure node_vectors_all_l5w3d100.txt is placed in the data/ or results/ directory.

4. Feature Construction (Part 3)
Construct the HIN-based feature vectors using the generated embeddings:

python src/construct_feature_vectors_3_HIN.py
5. Model Training and Prediction
Finally, run the prediction script to train the models (Random Forest, SVM, ElasticNet) and evaluate performance:

python src/prediction.py
Alternatively, you can use src/algorithm_123.py for specific algorithm variants.
